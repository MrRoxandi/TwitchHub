# HadrwareLib

## Доступные функции и объекты

Клавиатура

| Имя               | Аргументы                  | Описание                                                                                         |
| ----------------- | -------------------------- | ------------------------------------------------------------------------------------------------ |
| `keycodes`        | -                          | Таблица со всеми возможными клавишами доступными к эмуляции                                      |
| `parsekeycode`    | `string key`               | Функция переводит строковое значение в код клавиши                                               |
| `keycodetostring` | `int code`                 | Функция переводит код клавиши в строковое значние                                                |
| `keydown`         | `int code`                 | Функция симулирует нажатие клавиши по ее кодовому значению                                       |
| `keyup`           | `int code`                 | Функция симулирует отжатие клавиши по ее кодовому значению                                       |
| `keytype`         | `int code`                 | Функция симулирует краткое одиночное нажатие клавиши по ее кодовому значению                     |
| `keyhold`         | `int code`, `int duration` | Функция симулирует зажатие клавиши по ее кодовому значению на длительность равную `durations` мс |
| `typetext`        | `string text`              | Функция симулирует набор текста на клавиатуре (не поддерживает русский язык)                     |
| `keyisblocked`    | `int code`                 | Функция проверяет заблокирована ли клавиша                                                       |
| `keyblock`        | `int code`                 | Функция блокирует клавишу                                                                        |
| `keyunblock`      | `int code`                 | Функция разблокирует клавишу                                                                     |
| `keytoggle`       | `int code`                 | Функция переключает состоянеи блокировки клавиши                                                 |

Мышка

| Имя                | Аргументы                  | Описание                                                                               |
| ------------------ | -------------------------- | -------------------------------------------------------------------------------------- |
| `parsemousebutton` | `string button`            | Функция преобразует строкове значение клавиши мышки в код клавиши мышки                |
| `buttontostring`   | `int button`               | Функция преобразует кодовое значение клавиши мышки в строку                            |
| `mousedown`        | `int button`               | Функция симулирует нажатие клавиши мышки                                               |
| `mouseup`          | `int button`               | Функция симулирует отжатие клавиши мышки                                               |
| `mouseclick`       | `int button`               | Функция симулирует краткое одиночное нажатие клавиши мышки                             |
| `mousehold`        | `int button, int duration` | Функция симулирует зажатие клавиши мышки на длительность равную `duration` мс          |
| `scrollvertical`   | `int delta`                | Функция симулирует вертикальную прокрутку колесика мышки                               |
| `scrollhorizontal` | `int delta`                | Функция симулирует горизонтальную прокрутку колесика мышки                             |
| `setmouseposition` | `int x, int y`             | Функция устанавливает курсор в определённое положение в по координатам x, y            |
| `mousemove`        | `int dx, int dy`           | Функция перемещает курсор на определённое растояние относительно текущих координа x, y |
| `buttonisblock`    | `int keyCode`              | Функция проверяет заблокирована ли клавиша                                             |
| `buttonblock`      | `int keyCode`              | Функция блокирует клавишу                                                              |
| `buttonunblock`    | `int keyCode`              | Функция разблокирует клавишу                                                           |
| `buttontoggle`     | `int keyCode`              | Функция переключает состоянеи блокировки клавиши                                       |

Пример использования для симуляции нажатия левой кнопки мыши:

```lua
-- другой код
local code = hardwarelib:parsemousebutton('left')
hardwarelib:mouseclick(code)
```
