# LoggerLib

## Доступные функции

| Имя             | Аргументы                         | Описание                                                                                                       |
| --------------- | --------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `loginfo`       | `string message`                  | Функция для отправки лог-сообщения с уровнем логирования `info`                                                |
| `logdebug`      | `string message`                  | Функция для отправки лог-сообщения с уровнем логирования `debug`                                               |
| `logwarning`    | `string message`                  | Функция для отправки лог-сообщения с уровнем логирования `warning`                                             |
| `logerror`      | `string message`                  | Функция для отправки лог-сообщения с уровнем логирования `error`                                               |
| `loginfofmt`    | `string message`, `LuaTable args` | Функция для отправки лог-сообщения с уровнем логирования `info` с дополнительным форматированием аргументов    |
| `logdebugfmt`   | `string message`, `LuaTable args` | Функция для отправки лог-сообщения с уровнем логирования `debug` с дополнительным форматированием аргументов   |
| `logwarningfmt` | `string message`, `LuaTable args` | Функция для отправки лог-сообщения с уровнем логирования `warning` с дополнительным форматированием аргументов |
| `logerrorfmt`   | `string message`, `LuaTable args` | Функция для отправки лог-сообщения с уровнем логирования `error` с дополнительным форматированием аргументов   |

Пример использования для отправки форматированного сообщения уровня `info`

```lua
local arg = 'value`
loggerlib:loginfofmt("Some cool {0}", {arg})
-- Some cool value
```
