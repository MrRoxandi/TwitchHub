# StorageLib

## Доступные функции

| Имя        | Аргументы                     | Описание                                                                               |
| ---------- | ----------------------------- | -------------------------------------------------------------------------------------- |
| `load`     | -                             | Функция позволяет загрузить сохраненные данные из основного файла хранилища            |
| `save`     | -                             | Функция позволяет сохранить данные в основной файл хранилища                           |
| `backup`   | `string suffix`               | Функция позволяет сохранить данные в основной дополнительный (`suffix`) файл хранилища |
| `contains` | `string key`                  | Функция позволяет проверить наличие како-го элемента по ключу `key` в хранилище        |
| `count`    | -                             | Функция позволяет получить количество записей в хранилище                              |
| `keys`     | -                             | Функция позволяет получить все ключи существующих записей в хранилище                  |
| `get`      | `string key`                  | Функция позволяет получить запись (`LuaValue`) по ключу `key`                          |
| `set`      | `string key`,`LuaValue value` | Функция позволяет установить запись (`value`) по ключу `key`                           |
| `remove`   | `string key`                  | Функция позволяет удалить запись по ключу `key`                                        |
| `clear`    | -                             | Функция позволяет очистить хранилище                                                   |

Пример простого метода реализации заметок

```lua
local userid = '1298asdanj332' -- предполагаем, что это значение уникально

function callfn(userid, args)
  if #args == 0 then
    if storagelib:contains(userid) then
      return storagelib:get(userid)
    end
  else
    storagelib:set(userid, args)
  end
  return nil
end

callfn(userid, {'some text'})
local value = callfn(userid, {}) -- 'some text'
```
